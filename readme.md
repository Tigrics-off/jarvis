**1. Основные типы угроз:**

*   **Криминальные:** Кражи (карманники), мошенничество, физическое нападение.

*   **Техногенные:** Пожары, задымление, утечка газа, обрушения конструкций.

*   **Биологические/Экологические:** Массовые инфекции, химические выбросы.

*   **Социальные:** Стычки, паника, толпа (давки).



**2. Зоны повышенного внимания:**

*    Транспорт (метро, автобусы).

*   Торговые центры и вокзалы.

*   Парки и неосвещенные улицы.



**3. Меры предосторожности:**

*   **Контроль вещей:** Сумки держать перед собой, документы — в потайных карманах.

*   **Наблюдательность:** Сканирование выходов и поиск точек безопасности.

*   **Дистанция:** Избегать плотных скоплений людей (центр толпы — зона риска).

*   **Цифровая гигиена:** Осторожность с открытым Wi-Fi и личными данными.



**4. Алгоритм действий при ЧП:**

*   Сохранять спокойствие.

*   Двигаться по направлению к ближайшему выходу (не против движения толпы).

*   Связаться с экстренными службами (112).
